1 — Install Prisma
-->npm install prisma --save-dev

This installs the Prisma CLI (Command Line Tool).
-It is used for:
-npx prisma init
-npx prisma generate
-npx prisma db pull
-npx prisma migrate dev
-npx prisma studio
It helps you create schema, generate client, manage database.

Why --save-dev?
Prisma CLI is used only during development
It is NOT needed in production server

-->npm install @prisma/client
What is @prisma/client?
This is the actual database client used in your code.

import { PrismaClient } from "@prisma/client";
prisma.user.findMany()
prisma.user.findUnique()
prisma.task.create()

-->npm install prisma@6 @prisma/client@6
